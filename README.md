## Install

```sh
npm install
```
## Endpoints

```sh
GET /v1/match/bets/:matchid
GET /v1/user/bets/:username?inProgress=true
GET /v1/user/bets/:username
POST /v1/bet
PUT /v1/bet/resolve/:id
```
## Run tests

```sh
npm run test
```
